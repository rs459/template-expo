# Template Expo React Native

Ce template est configuré avec les meilleures pratiques et technologies modernes pour React Native.

## Stack Technique
- **Expo SDK (Latest)**
- **Expo Router**
- **NativeWind v4**
- **Zustand** + **Secure Store**
- **TanStack Query** + **Axios**
- **React Hook Form** + **Zod**
- **Jest** + **Testing Library**

## Structure
- `src/`: Code source (Composants, Ecrans, Services...)
- `app/`: Routes Expo Router

## Démarrage
### Utiliser comme template
Pour créer un nouveau projet en utilisant ce template (une fois hébergé sur GitHub) :
```bash
npx create-expo-app@latest mon-projet --template https://github.com/rs459/template-expo
```

### Développement local
1. `npm install`
2. `npm run ios` ou `npm run android`

## Tests
`npm test`
